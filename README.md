# 📒 Agenda de Contatos

Aplicação desenvolvida em **Python** utilizando **SQLite** para gerenciar contatos por meio de operações CRUD. Interface construída através do **Tkinter**.

## 🚀 Funcionalidades

* ➕ Adicionar contato
* 📋 Listar contatos
* 🔍 Buscar por nome
* ✏️ Atualizar contato
* 🗑️ Excluir contato

## 🛠️ Tecnologias

* Python 3
* SQLite3
* Tkinter

## ▶️ Como executar
1. Criar a tabela "contatos" executando o arquivo querys.py;
2. Executar

```bash
python interface.py
```

## 📚 Objetivo

Praticar o uso de Python com banco de dados SQLite, implementando um sistema CRUD em uma interface Tkinter.
